# 題庫專案工作規則

> 本檔為規則正本（Claude Code 讀這份；Codex 讀 AGENTS.md，內容相同）。
> 接手工作前，請再讀：`交接文件.md`（架構與進度）、`docs/圖文混合題庫解析規範.md`（解析細則）。

## 鐵律（不可變動）

1. **提供給你的題庫來源檔與答案，一律不可修改**：不改寫、不修正、不推測、不補完。
   來源檔目前是大考中心官方 PDF；未來可能是課本、月考考卷、模擬考考卷等（PDF 或照片），規則相同。
2. **答案一律取自來源答案檔**，完成後逐題核對，零誤差才算完成。
3. **內容有疑義**（模糊、抽不出、版面錯亂）→ 標記人工複核（review_flags），不可默默修字或猜測。
4. **非選擇題與作文暫不納入互動作答**（處理方案待傑克定案）。

## 題庫預設製作方式：「文字題＋必要裁圖＋題組共用資料」

- 純文字題盡量文字化（題幹、選項直接結構化）。
- 圖表、公式、插圖、無法可靠重建的版面（如雙欄），才裁來源檔局部圖。
- 題組的文章／表格／圖片只在題組層放一份，各題只存自己的題號、題幹、選項。
- 題目與答案完全依來源檔，不改、不猜。

（數學＝公式密集、屬「必要裁圖」比例極高的特例，可整題裁圖。範例：115 社會＝文字＋題組裁圖；115 數A＝整題裁圖。）

## 工作慣例

- **唯一主程式** `public/index.html`：題庫一律做成 `src/question-bank/{年}-{科}.js`（`QUESTION_BANKS.push` 註冊）並掛進主程式；不可另開獨立頁。
- **不建模擬題／測試分類**進主程式（2026-07-10 已全數清除，不再出現）。
- 每批做完先**本機驗收**（渲染、判分、手機 360px），回報：題數／題組數／裁圖數／待複核清單。
- **發佈閘門**：傑克確認前不 `git push`、不公開發佈。
- **公開版不含任何來源原檔**（PDF／掃描圖）；`.gitignore` 已排除。注意：官方試題依著作權法可重製，但課本、模擬卷等其他來源版權較嚴，公開發佈前要再跟傑克確認範圍。
- 專案資料夾只放：程式、題庫、開發過程紀錄、交接文件（＋本機來源檔）。
