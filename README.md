# 學測考試總複習

學生高三大考前使用的**學測考試總複習互動刷題工具**：答題卡作答、交卷自動判分、錯題與進度紀錄。獨立於 `vocab-app`（高中平時學習與月考前練習），兩專案互不影響。

- **線上使用**：https://jackchan1973.github.io/gsat-exam/

## 工作規則

規則正本在 [CLAUDE.md](CLAUDE.md)（Claude Code）／[AGENTS.md](AGENTS.md)（Codex），核心一句話：

> **提供的題庫來源檔與答案一律不可修改；題庫預設＝文字題＋必要裁圖＋題組共用資料。**

架構、產出流程與進度見 [交接文件.md](交接文件.md)；解析細則見 [docs/圖文混合題庫解析規範.md](docs/圖文混合題庫解析規範.md)。

## 題庫來源

目前：大考中心 113／114／115 學年度學測官方 PDF（來源原檔保存於不公開資料夾，**不上公開網站**）。
未來：課本、月考考卷、模擬考考卷等其他來源（PDF 或照片），製作規則相同。

官方來源頁：https://www.ceec.edu.tw/xmfile?xsmsid=0J052424829869345634
近 3 年官方 PDF 下載紀錄：`資料/大考中心官方PDF/近3年考題下載紀錄/README.md`

## 主要檔案與存放方式

- `public/index.html`：唯一主程式入口，所有互動題庫都掛在這裡。
- `src/app.js`：前端互動邏輯、答題卡流程、判分、進度紀錄。
- `src/style.css`：網站樣式與手機版面。
- `src/exam-files.js`：官方來源檔索引，提供年度、科目、檔案清單。
- `src/question-bank/{年}-{科}.js`：正式互動題庫，每科每年一個檔案。
- `public/images/{年}學年度/{科}/`：正式題庫需要用到的局部裁圖。
- `tools/`：抽取、裁圖、產生題庫的腳本。
- `data/raw-extract/`：官方來源檔抽出的中繼資料，不直接當正式題庫。
- `資料/大考中心官方PDF/`：官方 PDF 原始檔與下載紀錄，不進 git、不上公開網站。
- `docs/`：開發流程、解析規範、教學提示詞、單科歷史紀錄。
- `AGENTS.md` / `CLAUDE.md`：AI 工作規則正本，兩份要保持一致。
- `交接文件.md`：接手資訊、進度、架構與部署摘要。

## 文件分工

- `README.md`：專案入口、最新狀態、資料夾結構。
- `交接文件.md`：詳細接手資訊、架構、進度、部署與規則摘要。
- `AGENTS.md` / `CLAUDE.md`：Codex / Claude Code 的規則正本，兩份需保持內容一致。

- `docs/用AI開發刷題工具_提示詞與流程.md`：給老師或家長參考的通用教學文件。
- `docs/115數A_開發紀錄.md`：115 數學A 的歷史開發紀錄。
