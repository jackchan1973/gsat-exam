# 高三學測歷屆考題

高三學測總複習的**互動刷題工具**：答題卡作答、交卷自動判分、錯題與進度紀錄。獨立於 `vocab-app`（高一/高二段考複習），兩專案互不影響。

- **線上使用**：https://jackchan1973.github.io/gsat-exam/

## 工作規則

規則正本在 [CLAUDE.md](CLAUDE.md)（Claude Code）／[AGENTS.md](AGENTS.md)（Codex），核心一句話：

> **提供的題庫來源檔與答案一律不可修改；題庫預設＝文字題＋必要裁圖＋題組共用資料。**

架構、產出流程與進度見 [交接文件.md](交接文件.md)；解析細則見 [docs/圖文混合題庫解析規範.md](docs/圖文混合題庫解析規範.md)。

## 題庫來源

目前：大考中心 113／114／115 學年度學測官方 PDF（來源原檔保存於不公開資料夾，**不上公開網站**）。
未來：課本、月考考卷、模擬考考卷等其他來源（PDF 或照片），製作規則相同。

官方來源頁：https://www.ceec.edu.tw/xmfile?xsmsid=0J052424829869345634

## 目前題庫（2026-07-10）

已上線：115/114 國文、113/114 英文、115 數學A、115 社會、115 自然。
待做：115 數學B（「每科先做 115」收尾），之後回補其他年度。
手寫題（非選擇題、作文）：列出題目但不作答不判分（顯示功能陸續補上）。
