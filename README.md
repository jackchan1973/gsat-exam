# 高三學測歷屆考題

這是獨立於 `vocab-app` 的新專案，用來整理大考中心近年學測歷屆考題，未來發展成高三下刷歷屆、計時練習、錯題分析與五科學測複習工具。

## 專案定位

- `vocab-app`：目前主要服務高一/高二段考、課本、單字、文意選填、克漏字與課文複習。
- `高三學測歷屆考題`：高三學測總複習與歷屆試題刷題工具。

兩個專案分開存放，避免高三刷題資料和目前段考複習程式互相影響。

## 開發原則

- 這次開發先做單機版。
- 在使用者完整檢查與確認之前，不做公開發佈或正式部署。
- 本次開發是歷屆考題複習工具，所有題目與解答內容都必須維持官方原檔，不可改寫、修正、重製或自行更動答案。
- 測試階段可以使用清楚標示的模擬題測試流程，但不可把模擬題標示成官方歷屆題。
- 互動題庫以答題卡模式為主，優先收錄可劃卡作答的選擇題。
- 國寫作文題不做到本程式裡；作文 PDF 僅保留在官方 PDF 原檔瀏覽。
- 圖文分離、題組共用、表格合併與手機驗收的詳細規則見 [docs/圖文混合題庫解析規範.md](docs/圖文混合題庫解析規範.md)。

## 目前程式

第一版是純靜態網頁，可直接開啟：

```text
public/index.html
```

目前功能：

- 依年度篩選
- 依科目篩選
- 依檔案類型篩選
- 搜尋標題
- 開啟官方 PDF
- 顯示近三年學測資料總覽
