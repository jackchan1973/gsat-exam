> 最後更新：2026-07-10
> 本專案由 **Claude Code 與 Codex 協同開發**。後續接手者（Claude Code 或 Codex）請先讀**規則正本**（Claude 讀 `CLAUDE.md`／Codex 讀 `AGENTS.md`，內容相同），再讀本文件了解架構與進度。

## 1. 專案目的

獨立的「高三學測歷屆考題」互動刷題工具，跟 `vocab-app`（高一/高二段考複習）分開。定位是高三學測總複習：刷歷屆選擇題、自動對答案、錯題與弱點紀錄。

題庫來源檔目前是大考中心近三年（113/114/115）官方 PDF；**未來會有其他來源**（課本、月考考卷、模擬考考卷等，PDF 或照片），製作規則相同。

## 3. 唯一主程式與架構

**所有互動題庫都整合進單一主程式 `public/index.html`，不可另開獨立頁。**

```text
public/index.html            主程式（掛載各題庫 JS 與 app.js）
 ├─ src/exam-files.js        EXAM_FILES：來源檔索引（提供科目/年度清單）
 ├─ src/question-bank/*.js   各科各年題庫（QUESTION_BANKS.push(...) 各自註冊）
 ├─ src/app.js               SPA：科目→年度→分類→答題卡→師長檢查（進度存 localStorage）
 └─ src/style.css            樣式
index.html（根目錄）          自動導向 public/index.html（給 GitHub Pages 用）
```

SPA 流程：**選科目 → 選年度 → 選分類 → 答題卡練習 → 交卷自動判分 → 師長檢查（完成率／錯題／進度）**。

app.js 支援的題型呈現：純文字單選/多選、題組共用資料（文章 sticky／字庫／句庫／圖表 passage-image）、克漏字空格、圖片選項、整題圖片題＋(1)-(5) 選項、選填數字格、圖片題組（displayGroups）。手機 360px 已驗收。

## 4. 題庫產出流程

**預設製作方式：「文字題＋必要裁圖＋題組共用資料」**（詳見 CLAUDE.md／AGENTS.md）
- 純文字題盡量文字化（題幹、選項直接結構化）。
- 圖表、公式、插圖、無法可靠重建的版面，才裁來源檔局部圖。
- 題組的文章／表格／圖片只在題組層存一份，各題只存自己的題號、題幹、選項。
- 題目與答案完全依來源檔，不改、不猜。
- 數學＝公式密集特例（整題裁圖）。範例：115 社會（文字＋題組裁圖）、115 數A（整題裁圖）。

```text
來源檔（官方 PDF 等）──(tools/batch_extract.py)──▶ 中繼檔 data/raw-extract/{年}學年度/{科}/*.json
   （抽純文字/表格/圖片座標，只忠實搬運，不做判斷，省 token）
           │
           ▼
中繼檔 ──(build 腳本)──▶ src/question-bank/{年}-{科}.js（QUESTION_BANKS.push），在 index.html 掛一行 <script>
           │
           ▼
自動出現在主程式，走答題卡流程；答案取自來源答案檔，逐題核對零誤差才算數。
```

- 英文用**跨年度單一腳本** `tools/build_english.py {年}`：各年度差異只寫在腳本內 `CONFIG`。
- 需裁的圖輸出到 `public/images/{年}學年度/{科}/`，檔名帶題號或題組ID。

## 6. 製作順序

**每一科都先做 115 年那一份**（提早發現各科型態問題），再回補其他年度。國寫作文不做互動題庫。

## 7. 部署與版權

- GitHub Pages 從 `main` 分支發佈；更新＝`git push` 後約 1–2 分鐘生效（**推送前須經傑克確認**）。
- **公開版絕不含任何來源原檔**：`.gitignore` 已排除 `資料/大考中心官方PDF/`、`*.pdf`；前端無來源檔瀏覽功能。
- 來源檔只留在**本機**（`資料/大考中心官方PDF/` 等）。
- **版權分級注意**：官方考試試題依著作權法第 9 條可自由重製；但**課本、模擬考卷等其他來源有著作權**，未來這類題庫要公開發佈前，必須先跟傑克確認範圍（可能僅限私人使用）。

## 8. 鐵律與原則

**正本在 `CLAUDE.md`／`AGENTS.md`**，摘要：
1. 提供的題庫來源檔與答案一律不可修改（不改寫、不修正、不推測、不補完）。
2. 答案取自來源答案檔，逐題核對零誤差才算完成。
3. 有疑義標 review_flags 交人工複核，不猜測。
4. 非選擇題與作文暫不納入互動作答（方案待定）。
5. 題庫預設＝文字題＋必要裁圖＋題組共用資料。
6. 不建模擬題／測試分類；傑克驗收前不算正式完成、不發佈。
