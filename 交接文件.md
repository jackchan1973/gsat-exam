# 高三學測歷屆考題：交接文件

## 1. 專案目的

建立一個獨立的「高三學測歷屆考題」專案，專門處理大考中心學測歷屆試題，不和目前的 `vocab-app` 混在一起。

目前 `vocab-app` 的定位是高一/高二段考複習，重點在課本單字、文意選填、克漏字、課文句型與每日學習紀錄。

本專案的定位是高三學測總複習，尤其是高三下學期刷歷屆學測題、模擬卷、錯題與弱點分析。

## 7. 後續開發建議

### 第一階段：PDF 資料管理

- 補齊更多年度資料，例如 112、111、110。
- 加入「只看試題」與「只看答案」快速篩選。
- 每科建立固定入口：英文、數學A、數學B、國文、社會、自然。

### 第二階段：英文優先拆題

英文最適合先做成互動題庫，建議優先：

- 單字題
- 綜合測驗
- 文意選填
- 篇章結構
- 閱讀測驗

每題資料建議包含：

```js
{
  year: "115",
  subject: "英文",
  section: "綜合測驗",
  questionNo: 12,
  passage: "...",
  prompt: "...",
  options: ["A", "B", "C", "D"],
  answer: "B",
  explanation: "...",
  sourceFile: "..."
}
```

### 第三階段：刷題模式

- 按年度刷題。
- 按科目刷題。
- 按題型刷題。
- 混合隨機刷題。
- 計時模式。
- 作答紀錄。

### 第四階段：錯題與家長報告

- 錯題本。
- 弱點統計。
- 每日刷題量。
- 正確率。
- 近期進步曲線。
